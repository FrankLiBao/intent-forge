## Contributing

If you'd like to add documentation in other languages, please follow the structure used for the English documentation:

1. Create a new directory under `docs/` with the language code (e.g., `docs/ja/`, `docs/fr/`)
2. Translate the main documentation files
3. Update internal links to point to the correct paths
4. Add your language to this README

See [CONTRIBUTING.md](../CONTRIBUTING.md) (中文) or [CONTRIBUTING.md](en/CONTRIBUTING.md) (English) for contribution guidelines.
